# Twitter Clone Server  
Node.js, Express 기반 트위터 클론 서버 코딩
<br>

- 사이트 링크: [Twitter Clone Coding](https://tweet-clone-coding-server.onrender.com)  
<br>

## 주요 기능  
- JWT 기반 회원가입 / 로그인 기능  
- 로그인 성공 시 토큰 발급 및 인증 처리  
- MongoDB 연동을 통한 데이터 저장 및 조회  
- 전체 트윗 목록 조회  
- 트윗 작성 / 수정 / 삭제 기능  
- 내가 작성한 트윗만 필터링하여 조회
